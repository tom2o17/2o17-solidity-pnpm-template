# 🏎️ Foundry Template tom2o17 🏎️

Default Foundry Template with pnpm and no `.gitModules` 

## Usage 
### Install external contracts
```
pnpm i
```

### compile to bytecode 
```
pnpm build
```
#### or 
```
forge build
```

### Test said bytecode
```
pmpm test
```
#### or 
```
forge test
```

## Linter
### to check
```
pnpm lint:check
```

### to correct
```
pnpm lint:fix-all
```
